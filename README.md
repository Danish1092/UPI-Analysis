# UPI Analysis

This project analyzes UPI transactions for 2024 using Microsoft Fabric and Power BI.

## What I did

- Cleaned the dataset in Microsoft Fabric.
- Built the semantic model in Microsoft Fabric.
- Connected Power BI Desktop to the semantic model.
- Created visuals and published the final report.
